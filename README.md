# Telegram Shop Bot

Бот для продажи цифровых товаров в Telegram

## Особенности

- Система управления товарами
- Прием платежей через PayMaster
- Статистика продаж

## Установка

1. `pip install -r requirements.txt`
2. Переименовать файл '.env.example' в '.env'
3. Заполнить .env
4. Запустить `python main.py`

## Команды

- /start - начало работы
- /admin - админ-панель
